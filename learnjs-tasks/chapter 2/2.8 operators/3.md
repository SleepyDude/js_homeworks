# Преобразование типов
важность: 5
Какой результат будет у выражений ниже?
```js
"" + 1 + 0
"" - 1 + 0
true + false
6 / "3"
"2" * "3"
4 + 5 + "px"
"$" + 4 + 5
"4" - 2
"4px" - 2
"  -9  " + 5
"  -9  " - 5
null + 1
undefined + 1
" \t \n" - 2
```
Подумайте как следует, запишите ответы и сверьтесь с решением.

# Решение

```js
"" + 1 + 0 // '10' (бинарный плюс приводит к строкам, если есть строка)
"" - 1 + 0 // -1 (бинарный минус приведет операнды к числу)
true + false // 1 (в отсутствие строк приведет к числам)
6 / "3" // 2 (приведет к числу т.к. не "+")
"2" * "3" // 6 (приведет к числам)
4 + 5 + "px" // '9px' (первый плюс сложит как числа, второй - как строки)
"$" + 4 + 5 // '$45' (уже первый плюс вернет строку, что 'распространится' дальше)
"4" - 2 // 2 (не бинарный плюс => приведет к числу)
"4px" - 2 // NaN ('4px' не сможет привестись к числу корректно, результатом будет NaN)
"  -9  " + 5 // '   -9   5' ('заражение' строковостью при наличии бинарного плюса)
"  -9  " - 5 // -14 (бинарный минус => попытка привести первый операнд к числу, что будет удачным т.к пробелы отбросятся)
null + 1 // 1 (null приводится к числу 0)
undefined + 1 // NaN (undefined приводится к NaN, что делает NaN и дальнейшее выражение)
" \t \n" - 2 // -2 (попытка привести строку к числу успешна, отброшены все "пробельные" символы => пустая строка => приводится к 0 => 0-2 = -2)
```
